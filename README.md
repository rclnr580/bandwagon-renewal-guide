## 搬瓦工续费的基本机制（先搞清楚再操作）

搬瓦工的续费不是按钮点一下就能提前搞定的那种——它有一套固定的时间节点，理解了这个，后面的操作才不慌。

**续费账单的生成时间：VPS 到期当天。**

不是提前一周，不是到期前三天，就是到期那天系统自动生成一张续费账单（Invoice），同时向你的注册邮箱发一封标题为 "Invoice #XXXXX Generated" 的邮件。

**账单有效期：到期后 7 天。**

这 7 天里你的 VPS 还在正常跑，不影响使用。第 7 天结束后还没付，账单状态变 Cancelled，VPS 被删除，数据清空，无法恢复。

关键点在这：搬瓦工不会自动从你的信用卡或支付宝扣款，官方说得很明确——"we never charge your payment method automatically"。所以账单出来了也得你自己去付。

## 搬瓦工未收到续费邮件的三个原因

收不到邮件，基本就这三种情况，逐一排查。

**原因一：邮件进了垃圾箱**

这是最常见的。国内很多邮箱（163、QQ、新浪）会把搬瓦工发来的邮件直接扔进 Spam 文件夹，你主收件箱里什么都没有，以为没到期。

处理方法：登进邮箱，去垃圾邮件里搜 "bandwagonhost" 或 "Invoice"，大概率能翻出来。找到了直接点邮件里的付款链接就行。

**原因二：注册邮箱填错了**

注册的时候手抖，或者邮箱地址换了没更新。账单发出去，根本就不是发到你实际在用的邮箱。

确认方法：登录搬瓦工客户端后台 → 右上角账号 → Edit Account Details → 看 Email Address 那栏填的是什么。

**原因三：手动关闭了续费通知**

这个相对少见，但存在。有些人当时嫌邮件太多把通知关了，关了就没账单、没邮件，到期就直接删了。

确认方法：后台 → My Services → 对应 VPS → 看续费开关是否 ON。

## 没收到邮件也能续费——后台操作路径

找不到邮件不要慌，后台直接操作就行，不需要依赖那封邮件。

1. 登录搬瓦工客户端：bwh81.net 或 bwh88.net（大陆可以直连）
2. 顶部菜单 → **Billing** → **My Invoices**
3. 找到状态为 **Unpaid（未支付）** 的账单，点右边的 **View Invoice**
4. 进入账单页面后点 **Pay Now**，选择支付宝或其他方式完成付款

付款成功后，VPS 续费周期立即延长，后台状态会从 Unpaid 变成 Paid。

**注意一个细节：** 如果 My Invoices 里没有 Unpaid 账单，有两种可能——一是账单还没生成（VPS 还没到期），二是账户余额已经自动扣款完成了。这两种情况都不用担心，前者等到期当天再查，后者说明自动续费已经跑了。

## VPS 已被删除，还能救回来吗？

说实话，这个概率很低，但不是完全没有。

**情况 A：VPS 还在运行但账单是 Unpaid 状态**

这说明 7 天宽限期还没过，赶紧去后台付账单，付完立刻恢复正常，数据一点没丢。

**情况 B：VPS 已被暂停（Suspend）但账单还在**

暂停不等于删除。账单还是 Unpaid 状态的话，付完款服务通常会恢复。

**情况 C：VPS 已被删除，账单状态变成 Cancelled**

这种情况没有恢复手段。数据已经被清空，账单也作废了，唯一的路是重新购买一台新的 VPS。

所以抢在账单失效前付款是关键，别拖。

## 搬瓦工历史邮件在哪查

有个功能很多人不知道：搬瓦工后台可以查看所有发送过的历史邮件，不依赖你的邮箱有没有收到。

路径：登录后台 → 右上角账号菜单 → **Email History**

所有账单邮件、系统通知都在这里，方便直接找到续费链接或账单编号。

## 怎么核对自己的续费日期

与其等邮件，不如主动查。

登录后台 → **My Services** → 找到对应 VPS，**Next Due Date** 那一列就是下次续费日期。这个日期比任何邮件都可靠，建议买了之后直接截个图或者在手机日历里设个提醒，比等邮件靠谱得多。

## 常见问题解答

**Q：搬瓦工支持提前续费吗？**

不支持。账单只在到期当天生成，在那之前后台不会出现续费按钮。如果你想提前"锁定"，唯一的方式是往账户里预充值，等账单自动出来时系统扣款。

**Q：续费邮件里的链接失效了怎么办？**

直接去后台操作就行，不依赖邮件链接。进 Billing → My Invoices，找 Unpaid 账单，点 Pay Now，效果是一样的。

**Q：续费支付宝付款后，后台还是显示 Unpaid，怎么回事？**

支付宝有时会有延迟，先等几分钟刷新一下。如果超过 30 分钟还没变，可以进后台提交一个工单，附上支付宝的付款截图，客服处理速度还可以，通常当天会处理。

**Q：搬瓦工未收到续费邮件是不是账号有问题？**

大概率不是账号问题，主要就是前面说的那三种情况：进垃圾箱、邮箱填错、通知被关。先从垃圾箱查起，再核对邮箱地址，90% 的情况都能在这两步解决。

**Q：VPS 删了之后数据还能找回吗？**

没有办法恢复。搬瓦工在删除时会把数据全部清空，这是行业惯例，不是搬瓦工的特殊政策。防止这种情况的唯一手段是：定期备份 + 不让账单过期。搬瓦工后台的 KiwiVM 控制面板有快照功能，可以手动备份；如果用到了重要数据，建议每周或每月备份一次。
